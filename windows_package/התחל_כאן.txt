     (סמן ✓ "Add Python to PATH" בהתקנה!)
   
   ✓ Inno Setup → https://jrsoftware.org/isdl.php
     (צריך רק אם רוצה ליצור קובץ התקנה מקצועי)

שלב 2: הרץ בניה
   ⚡ לחץ פעמיים על: build_all_windows.bat
   ⏱ המתן 2-5 דקות

שלב 3: התוצר
   📦 installer_output\PDF_Processor_Setup.exe
   
   זהו! עכשיו תוכל להפיץ את קובץ ההתקנה הזה 
   לכל משתמש Windows - הוא יתקין הכל אוטומטית.
